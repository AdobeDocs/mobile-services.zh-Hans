# Making calls to the library{#making-calls-to-the-library}

当您要从脚本调用插件时，请导入命名空间：

* **C#：**&#x200B;使用 `com.adobe.mobile;`

* **JavaScript：**&#x200B;导入 `com.adobe.mobile;`

* **boo：**&#x200B;导入 `com.adobe.mobile;`

导入命名空间后，可以通过ADBMobile类的静态方法直接调用插件。
