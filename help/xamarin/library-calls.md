# 调用库{#making-calls-to-the-library}

此信息可帮助您从脚本中调用插件。

如果要从脚本中调用插件，则必须导入命名空间。

使用 `Com.Adobe.Mobile`:

* **iOS**:导入命名空间后，您可以通过 `ADBMobile` 类。

* **Android**:您可以在 `Config/Analytics/Target/AudienceManager/Media`类。
