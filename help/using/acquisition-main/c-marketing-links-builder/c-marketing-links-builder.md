# 营销链接生成器{#marketing-links-builder}

生成器可帮助您生成营销链接，以将用户路由到应用商店和应用程序或 Web 链接目标。营销人员可基于规则选择目标，如用户设备的设备类型或操作系统。

营销链接可被跟踪和报告，以允许营销人员将下载、应用程序重新启动及其他应用程序内事件归因于这些链接。
