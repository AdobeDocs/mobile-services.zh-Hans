# 营销链接生成器{#marketing-links-builder}

构建器可帮助您构建可将用户引导到应用商店、应用程序或Web链接目标的营销链接。营销人员可以根据规则(如设备类型或用户设备的操作系统)选择目标。

可以跟踪和报告营销链接，这允许营销人员将下载、应用程序rela撤消以及其他应用程序内事件与这些链接进行整合。
